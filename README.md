# Joke or Fake Game

A simple browser-based game where users guess whether a joke is **real (from an API)** or **fake (manually created)**.

---

## Demo

Click the button, read the joke, and guess:

* ✅ Real
* ❌ Fake

Get instant feedback and try again!

---

## How It Works

* Fetches real jokes from the **Official Joke API**
* Randomly mixes in fake jokes
* User guesses → app checks correctness
